static float MathHelper.Mathematics.Perlin1D ( float x,
uint seed = 0,
FadeDelegate fade = null )
static

Generates a 1-dimensional noise value based on the input coordinate and seed using gradient hashing.

Parameters
xThe input coordinate in 1D space.
seedAn optional seed value to control the random gradients. Default is 0.
fadeA delegate to a fade function used to smooth the interpolation of values. If null, a default fade function will be used.
Returns
A float value representing the generated noise at the input coordinate.

This method uses a gradient hashing technique to compute the noise value at a given point in 1D space. A fade function is applied to smooth the interpolation of values.
