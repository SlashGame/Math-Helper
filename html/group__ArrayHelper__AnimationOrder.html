Generates a collection of diagonals from a 2D array starting from specified axes.

Template Parameters
TThe type of the elements in the 2D array.
Parameters
arrayThe 2D array from which diagonals will be generated.
starX0Indicates whether to start the diagonals from x-axis at 0.
starY0Indicates whether to start the diagonals from y-axis at 0.
Returns
A collection of diagonals where each diagonal is represented by an array of Vector2Int, indicating the indices of elements along that diagonal.

This IEnumerable is primarily used for generating animations.
