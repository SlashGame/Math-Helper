MathHelper.VectorHelper Class Reference

The VectorHelper class provides a series of utility methods for processing vectors in Unity. It includes functionalities for comparing vector magnitudes, applying mathematical operations such as floor, ceiling, rounding, and clamping, as well as converting vectors to arrays and masking specific components. Detailed Description

The VectorHelper class provides a series of utility methods for processing vectors in Unity. It includes functionalities for comparing vector magnitudes, applying mathematical operations such as floor, ceiling, rounding, and clamping, as well as converting vectors to arrays and masking specific components.
