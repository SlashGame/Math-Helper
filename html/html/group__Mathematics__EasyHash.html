static uint MathHelper.Mathematics.FNV1aHash ( string data)
static

Computes the FNV-1a hash for the given string.

Parameters
dataThe input string to compute the hash for.
Returns
The 32-bit FNV-1a hash of the input string.

FNV-1a (Fowler-Noll-Vo) is a non-cryptographic hash function that is fast and provides good distribution. This implementation uses the 32-bit variant with the following parameters:

  • FNV prime: 0x811C9DC5
  • FNV offset basis: 0x01000193 Note that this hash function is not suitable for cryptographic purposes.

Computes a hash for the given three integer coordinates using Jenkins one-at-a-time hash with an optional seed.

Parameters
xThe x-coordinate.
yThe y-coordinate. Default is 0.
zThe z-coordinate. Default is 0.
seedAn optional seed value for controlling the hash output. Default is 0.
Returns
An integer hash value calculated from the input coordinates and seed.

This method implements the Jenkins one-at-a-time hash function, which is designed to provide a good distribution of hash values for small inputs. It allows an additional seed to control hash output and does not require large prime numbers or additional libraries.
