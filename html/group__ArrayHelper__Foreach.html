◆ DoSomethingAction< T >()

delegate void MathHelper.ArrayHelper.DoSomethingAction< T > ( ref T item,
params int[] index )

Represents a method that performs an action on an element within a collection, providing both the element and its corresponding indices.

Template Parameters
TThe type of the elements in the collection.
Parameters
itemA reference to the element on which the action is performed.
indexThe indices that represent the position of the element within the collection. For multi-dimensional arrays, all indices are provided as a params array.
